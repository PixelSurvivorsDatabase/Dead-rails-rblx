# Dead Rails RBLX

A Rojo-powered Roblox Studio project for a Dead Rails-style survival/train game workflow test.

## Local setup

1. Clone this repo:

```bash
git clone https://github.com/PixelSurvivorsDatabase/Dead-rails-rblx.git
cd Dead-rails-rblx
```

2. Install Rojo and the Roblox Studio Rojo plugin.

3. Start Rojo from the repo folder:

```bash
rojo serve
```

4. Open Roblox Studio, open the Rojo plugin, and connect to the running server.
